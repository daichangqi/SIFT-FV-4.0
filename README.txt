READ ME LFW-Fisher_Vector_Program

Persiapan:
1. Download Dataset LFW di link: http://vis-www.cs.umass.edu/lfw/lfw.tgz
2. Buat base folder di pc anda (eg: YOUR_FOLDER)
3. Extract folder lfw pada sebuah folder (eg: YOUR_FOLDER->lfw)
4. Download dan Install Matlab R2015a
5. Download toolbox vl_feat untuk matlab di link: http://www.vlfeat.org/download/vlfeat-0.9.20-bin.tar.gz
6. extract toolbox vl_feat di base folder (eg: YOUR_FOLDER->vlfeat)
7. copy dan paste kode program pada folder PROGRAM ke base folder anda
8. download file .mat didalam folder [bigfiles] pada link: http://bit.ly/LFW_5112100086
9. letakkan folder [bigfiles] pada pc anda (bisa pada base folder atau tidak)

Running:
1. Jalankan program startup.m menggunakan Matlab lalu pilih lokasi base folder untuk code dan lokasi folder [bigfiles]
2. Jalankan program main line-by-line (atau running satu program langsung)
3. Apabila menemui masalah, pastikan alamat lokasi file pada program sesuai dengan di folder PC anda.
